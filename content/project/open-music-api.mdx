## Features

<ul>
  <li>
    Auth : authentication and authorization using JWT and using BFF (Backend for
    frontend)
  </li>
  <li>Song : Create, Get, Edit, and Delete Song by Artist</li>
  <li>
    Album: Create, Get, Edit Album based on Songs, upload Album Cover, add Likes
    on Album
  </li>
  <li>
    Playlist : Creating, Get, Edit Playlist, and Add or Remove song from
    Playlist
  </li>
  <li>
    Playlist Activities: Activites on Editing Playlist and add or removing song
    to the Playlist
  </li>
  <li>Collaboration: add,remove Collaboration on editing Playlist</li>
  <li>
    Export : Exporting Playlist to the email using RabbitMQ. consumer app is{" "}
